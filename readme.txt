== Description ==

MediaPress Media View Counter is a simple addon for MediaPress that allows counting and showing the total number of views for any media. As a site admin you can choose whether to count the media uploaderâ€™s own view or not.
This plugin can be used in conjunction with other plugins to show top media by views or the most popular media.

**For Support**, Please use [BuddyDev Support Forums](https://buddydev.com/support/forums/ ).

== Installation ==

1. Visit Dashboard->Plugins->add New
2. Search for MediaPress - Media View Counter
3. Install this plugin
4. Click Activate
5. Now, save settings under MediaPress->settings->addons.

For more details, Please see the plugin documentation page.
